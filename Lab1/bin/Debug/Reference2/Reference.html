Лабораторная работа №4 "Конечные автоматы"

Цель работы: Изучение конечных автоматов.
В соответствии с заданным вариантом задания:

  1. Спроектировать граф конечного автомата.
  2. Выполнить программную реализацию алгоритма работы конечного автомата

В качестве варианта задания было взято задание из курсовой работы: декларирование переменных с индексами языка FORTRAN.

Описание грамматики:
V(T) = {real, complex, integer, logical, character, ::, 1, 2, 3,..., 9, a, b, c,...Y, Z}
V(N) = {<ТИП>, <ИМЯ>, <ПАРАМЕТР>,<БУКВА>,<ЦИФРА>}
P = {
1) Z -> type '::' <ИМЯ><ПАРАМЕТР>
2)type - <ТИП> -> {real|complex|integer|logical|character}
3)<ИМЯ> -> letter{letter|digit}
4)<ПАРАМЕТР> -> '('<НАЧАЛО_ПАРАМЕТРА>
5)<НАЧАЛО_ПАРАМЕТРА> -> digit<КОНЕЦ_ПАРАМЕТРА>
6)<КОНЕЦ_ПАРАМЕТРА> -> ')'|digit<КОНЕЦ_ПАРАМЕТРА>
7)letter - <БУКВА> ->{a|b|c|...|Y|Z}
8)digit - <ЦИФРА> -> {1|2|3|...|8|9}
